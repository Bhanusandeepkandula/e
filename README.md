# 📅 Event Planner App

A sleek and powerful web-based event planning application to manage all your events, track payments, and stay organized with real-time status updates and calendar integration.

🔗 **Live Demo:** https://bhanusandeepkandula.github.io/e/

---

## 🚀 Features

- 🔐 Secure login via 4-digit PIN (Admin/User roles)
- ➕ Add, edit, delete, and view events with full detail
- 🎨 Responsive UI with both table and mobile card views
- 📊 Real-time dashboard with event counts and financial stats:
  - Total Earned
  - Balance Due
  - Event Statuses: Scheduled, Ongoing, Completed, Cancelled
- 🔍 Search & filter events by customer name, phone number, and status
- 📧 Email notifications with styled HTML templates on:
  - Event Creation
  - Updates
  - Deletions
  - Reminders
- 📅 Add to Calendar via downloadable `.ics` files
- 📥 Export event data as CSV
- 📌 Admin-only permissions for edits/deletes
- 💡 Smart insights on upcoming events (next 48 hours)

---

## 🛠 Tech Stack

- **Frontend:** HTML, CSS (TailwindCSS), JavaScript
- **Backend (Realtime DB):** Firebase
- **Email Service:** EmailJS
- **Calendar Export:** `.ics` generation (iCalendar format)
- **Charts:** Chart.js (Revenue breakdown - optional)
